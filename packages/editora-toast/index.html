Editora Toast · v2.x

Notifications that
don't get in the way.

A zero-dependency toast library with spring physics, promise lifecycle, 13 themes, swipe-to-dismiss, RTL, and full accessibility — shipped in <8 KB gzipped.

npm npm install @editora/toast
Getting started
Toast types
Four semantic levels plus loading and promise, all with auto-dismiss.
Fire a toast
import { toast } from '@editora/toast'; import '@editora/toast/toast.css'; toast.success('File saved.'); toast.error('Upload failed.'); toast.warning('Storage almost full.');
Async
Promise lifecycle
Pass any promise and get automatic loading → success / error transitions.
toast.promise(uploadFile(file), { loading: 'Uploading…', success: 'Upload complete!', error: 'Upload failed. Try again.' });
Placement
Positions
Six anchors plus center. Can be set globally or per toast.
Appearance
Themes
Thirteen built-in themes. Override any CSS variable for a fully custom look.
Active: light
Advanced
More features
Stacking, custom content, RTL, swipe-to-dismiss, and plugin hooks.
Rich interactions
Configuration
toast.configure({ position: 'bottom-right', theme: 'light', duration: 4000, maxVisible: 5, pauseOnHover: true, swipeDismiss: true, });
