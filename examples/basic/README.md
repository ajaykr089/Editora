# Basic Example

Simple Editora Rich Text Editor setup with minimal configuration.

## Features

- Bold, italic, underline, strikethrough formatting
- Headings (H1-H6)
- Lists (bullet and numbered)
- Links
- Undo/redo

## Running the Example

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Code Overview

The example demonstrates:

1. **Plugin Setup**: Initialize essential plugins
2. **State Management**: Use React state for content
3. **Real-time Output**: Display HTML output as you type

## Files

- `src/App.tsx` - Main application component
- `src/main.tsx` - Entry point
- `vite.config.ts` - Vite configuration
- `package.json` - Dependencies
