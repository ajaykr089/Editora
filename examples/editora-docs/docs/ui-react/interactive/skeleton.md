# Skeleton

The `Skeleton` component provides loading states that reduce layout shift.

## Basic Usage

```tsx
import { Skeleton } from '@editora/ui-react';

export function Example() {
  return <Skeleton style={{ width: 240, height: 16 }} />;
}
```
