# Separator

The `Separator` component separates related groups with semantic and visual clarity.

## Basic Usage

```tsx
import { Separator } from '@editora/ui-react';

export function Example() {
  return (
    <>
      <div>Primary actions</div>
      <Separator />
      <div>Secondary actions</div>
    </>
  );
}
```
