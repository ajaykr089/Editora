# PluginPanel

The `PluginPanel` component is used for plugin-side controls in editor layouts.

## Basic Usage

```tsx
import { PluginPanel } from '@editora/ui-react';

export function Example() {
  return <PluginPanel>Plugin settings</PluginPanel>;
}
```
