# Flex

```tsx
import { Flex } from '@editora/ui-react';

<Flex direction={{ initial: 'column', md: 'row' }} gap="1rem" align="center">
  <div>One</div>
  <div>Two</div>
</Flex>;
```

## Key Props

`direction`, `align`, `justify`, `wrap`, `gap`, `rowGap`, `columnGap`, `display`, `headless`
