Editora — Core vs Full Bundles

A quick comparison for vanilla HTML usage (CDN or local bundles).

Core Bundle ~96 KB

What it includes

  • Essential editing features (formatting, lists, history)
  • Small footprint for fast initial load
  • Ideal for lightweight integrations
Try the lightweight core editor.

Full Bundle ~1.9 MB

What it includes

  • All plugins (tables, images, math, export, media manager)
  • Feature-rich UI and dialogs
  • Best for applications that need full functionality
Try the full-featured editor.

Quick CDN Usage

Include either the Core or Full bundle via CDN. Core should be loaded before Full if you want to compare side-by-side.

<!-- Core bundle (smaller) -->
<script type="module" src="https://cdn.jsdelivr.net/npm/@editora/core/dist/webcomponent-core.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@editora/core/dist/webcomponent-core.min.css">

<!-- Full bundle (all plugins) -->
<script type="module" src="https://cdn.jsdelivr.net/npm/@editora/core/dist/webcomponent.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@editora/core/dist/webcomponent.min.css">
