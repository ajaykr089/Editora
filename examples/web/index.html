‚ö° Quick Start in 3 Minutes

1. Install

npm install @editora/core @editora/react

Or use via CDN for HTML websites without build tools.

2. Import

import { Editor } from '@editora/core';
// Or React
import RichTextEditor from '@editora/react';

3. Use

const editor = new Editor({
  element: '#editor',
  content: '<p>Hello!</p>'
});
